# Educa.AI

Plataforma inteligente para apoiar professores do ensino público
na criação de conteúdos pedagógicos com uso de Inteligência Artificial.

## Tecnologias Utilizadas
- Front-end: React.js
- Back-end: Node.js com Express
- Banco de Dados: MongoDB (simulado no MVP)
- Integração de IA: API de linguagem natural (mock)
- Hospedagem: Vercel (frontend) e Render (backend)

## Arquitetura do Sistema
- Interface Web (Professor)
- API Backend
- Serviço de IA para geração de conteúdo
- Banco de Dados

## MVP – Funcionalidades
- Cadastro e login de professores (simulado)
- Criação automática de atividades
- Geração de quizzes (proposto)
- Organização por turma (proposto)
- Visualização e compartilhamento de conteúdos

O MVP demonstra claramente o funcionamento do sistema,
mesmo com algumas funcionalidades simuladas.
